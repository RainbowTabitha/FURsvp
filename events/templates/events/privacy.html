Privacy Policy

Last modified: July 31, 2025


Your privacy is important to us. This policy explains how FURsvp collects, uses, and protects your information.

1. Data Storage and Security

  • FURsvp uses SQLite3 as its database system for secure and efficient data storage.
  • All data is encrypted and access is strictly controlled.
  • Regular backups and security audits are performed to protect your information.

2. Unified Platform

  • FURsvp now operates as a single, unified platform for all users.
  • All user data is securely stored in a centralized database.
  • You only need one account to access FURsvp, regardless of your location.
  • Data is never shared with third parties and remains protected within the platform.

3. Data Collection and Usage

  • We collect account information (username, email), event details, RSVP information, group memberships, and communication preferences.
  • This data is used to provide and improve our services, manage events, and send important notifications.
  • For users who connect their Telegram accounts, we store Telegram user IDs to enable platform integration features.

4. Third-Party Integrations and APIs

Telegram Integration

  • FURsvp uses Telegram's official APIs to provide integration features with the platform.
  • The FURsvp_bot is used exclusively for posting event updates and notifications to Telegram channels and groups.
  • The bot does not collect, store, or process any personal user data beyond what is necessary for posting functionality.
  • All other data processing and storage is handled directly by FURsvp's own systems.
  • Telegram user IDs are stored locally to enable platform integration features, but no other Telegram data is collected.

RSSHub Integration

  • FURsvp uses a custom fork of RSSHub hosted at https://rss.tabithahanegan.com to fetch and process RSS feeds.
  • This service is used to aggregate content from various sources for the blog and news features.
  • The RSSHub instance is privately hosted and does not store or process any FURsvp user data.
  • All data processing occurs on FURsvp's own servers; RSSHub is used solely as a data source.

5. Data Protection

  • Passwords are securely hashed using industry standards.
  • Access to your data is restricted to authorized personnel only.
  • We follow best practices for web security and data protection.
  • All API integrations are secured using industry-standard authentication and encryption.

6. Data Retention

  • Your data is retained as long as your account is active.
  • When you delete your account, personal information is removed; some event data may be retained for legal or operational reasons.
  • Backup copies are kept for a limited time.

7. Your Rights

  • Access, correct, or delete your personal data at any time.
  • Export your data in a portable format.
  • Opt out of non-essential communications.

8. Open Source and Technology

  • FURsvp is open source and available on GitHub under the MIT License.
  • Built with Django, a secure and reliable web framework.
  • Open source code does not grant access to user data; each instance's data remains private and isolated.
  • All privacy practices described here apply to both official and self-hosted instances.

Note: Running your own instance of FURsvp does not provide access to data from other instances.

9. Updates to This Policy

This privacy policy may be updated periodically. We will notify users of any significant changes.

10. Contact

If you have any questions about our privacy policy or data practices, please contact us at info@fursvp.org.
